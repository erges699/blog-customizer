# Проектная работа: Blog-customizer

### Страница блога с настраиваемым через CSS-переменные дизайном

Кастомизация страницы блога позволяющая выбрать:

- шрифт текста,
- размер шрифта,
- цвет шрифта,
- цвет фона,
- ширина контента.

## Запуск проекта

```
npm i
npm run build
npm run start
```

Для запуска storybook:

```
npm run storybook
```
